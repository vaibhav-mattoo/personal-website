# Public documents

Drop public-facing PDFs here. Each file you want to surface on
`/notes/documents/` also needs a matching entry in
`site/src/config/public-documents.ts`.

Files placed here without an entry in the config WILL still be
accessible at `https://vmattoo.dev/documents/<filename>` by anyone who
knows the URL. The config just controls what shows up in the listing.

For privacy: only drop files here that are intentionally public. Use
the private Stirling-PDF instance (`pdfs.vmattoo.dev`) for everything
else.
